# Dự báo giá xe máy cũ & phát hiện tin đăng bất thường

## 1. Mô tả dự án
Dự án xây dựng mô hình dự báo giá xe máy cũ trên dữ liệu Chợ Tốt,
kết hợp phát hiện bất thường (anomaly detection).

## 2. Công nghệ sử dụng
- Python, Pandas, Sklearn, PySpark
- XGBoost
- Streamlit (giao diện web)
- Heroku (triển khai)

## 3. Cấu trúc thư mục
- train_sklearn_motorbike_model.py — Train mô hình Sklearn  
- xay_dung_mo_hinh.py — Xử lý dữ liệu PySpark  
- demo_streamlit.py — App giao diện Streamlit  
- motorbike_cleaned.xlsx — Dữ liệu đã xử lý
